Router (Tools ein-/ausblenden)

Stufenweise Erkundung: In tools[] ist immer nur der Katalog (kurz). Volles Parameter-Schema kommt nur aus der Antwort von get_tool_help(tool_name) — nach list_tool_categories → list_tools_in_category. Router begrenzt nur, welche Tools in der Liste sind (letzte User-Nachricht + Trigger); ohne Treffer = alle Tools, ebenfalls nur Katalog.

Zusätzliche Kategorien nur explizit: Header X-Agent-Router-Categories: tool_factory (Proxy/Client) oder JSON "agent_router_categories": ["tool_factory"] im Chat-Body, falls der Client durchreicht. Open WebUI setzt solche Header meist nicht.
